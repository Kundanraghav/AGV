tcpdump - the packet capture tool
-r /tmp/traffic.pcap - read from the existing capture file (instead of capturing live)
'port 80' - filter to show only packets on port 80 (HTTP traffic)
-A - print packet contents in ASCII format (human-readable text)
| strings - pipe output to strings command, which extracts only printable text (filters>
| grep "14ce18" - pipe to grep to search for lines containing "14ce18"
